# ssus
Simple SCU UGR Scrapper 

## How to use?

Install it from NPM as follow:

```
    $ npm i ssus
```

Use it as follow:

```js
    const ssus = require('ssus');
    ssus()
        .then((json) => console.log(json))
        .catch((err) => console.error(err));
```
